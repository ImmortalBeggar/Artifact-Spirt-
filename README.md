# 🔮 Aroxia: The Open-Source Artifact Spirit

Aroxia is a high-performance Telegram AI bot designed for stability, low-cost hosting (Termux/VPS), and community-driven energy management.

## 🌌 Key Features
- **Triple-Tier Brain:** Automatically switches between Gemini 2.0 and Groq Llama-3 if a core fails or hits a quota limit.
- **Sect Treasury:** Allows group members to "donate" API keys to power the bot's communal energy.
- **Artifact Scanning:** Analyze documents and images (Powered by Gemini Vision).
- **Automated Ritual:** Single-command installation for Termux and Linux.
